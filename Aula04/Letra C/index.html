Utilizando a mesma estrutura de 3 arquivos (HTML+CSS+JS):
A inserção de um valor numérico não inteiro; Clicar em um botão que ao ser clicado exiba a informação do maior número inteiro menor do que o valor inserido pelo usuário e a informação do menor número inteiro maior do que o valor inserido pelo usuário. Observação: verificar se o valor inserido pelo usuário é do tipo numérico e, caso contrário, mostre ao usuário uma mensagem de erro.
